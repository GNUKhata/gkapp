## Quick Start

#### Get the repo

* [Clone from Gitlab](https://gitlab.com/gnukhata/gkapp)

#### Install

`cd` to project's dir and run `npm install` 

#### Serve

To pre-compile & hot-reload for development run `npm run serve`

#### Build

Production-ready with minified bundle `npm run build`
